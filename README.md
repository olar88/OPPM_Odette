# One Page Project Management System

一個基於 React + TypeScript 的單頁式專案管理系統，支援任務管理、資源分配、成本追蹤與進度視覺化。

## 功能特色

### 🏠 首頁導航
- 創建新專案
- 載入已儲存的專案
- 專案搜尋與管理

### 📊 專案管理核心功能
- **任務管理**: 新增、編輯、刪除任務，支援工期設定
- **資源分配**: 人力資源管理與費率設定
- **時間軸追蹤**: 視覺化專案時程與資源分配
- **成本計算**: 即時計算專案成本與累計金額
- **問題管理**: 記錄與追蹤專案問題

### 🎨 設計特點
- **Morandi 色系**: 舒適配色方案
- **響應式設計**: 支援桌面與行動裝置

### 💾 資料持久化
- **本地儲存**: 提供本地暫存
- **專案匯出**: PDF 報表匯出功能
- **資料備份**: 專案資料的儲存與載入

## 技術棧

- **Frontend**: React 18 + TypeScript
- **樣式**: Tailwind CSS 3.4.0
- **圖示**: Lucide React
- **資料儲存**: IndexedDB
- **建置工具**: Create React App
